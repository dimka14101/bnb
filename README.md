# BNB
Mobile system for blind people
# Example & prototype
Working prototype is located here: bnbproject.azurewebsites.net
At the bottom of the prototype page there is a link to the API documentation for writing your own client
# Installing application
1. Download BNB.apk file to you android device
2. Install downloaded BNB.apk
3. Give permission for all necessary permissions 
4. Press start button..
